## Slide Design Tips for Non-Designers

### Color Scheme (ESGLend Brand)
- **Primary Green:** #2e7d32 (dark green for headers)
- **Secondary Green:** #4caf50 (medium green for accents)
- **Light Green:** #66bb6a (for backgrounds)
- **White:** #ffffff (text on green backgrounds)
- **Dark Gray:** #212121 (body text)
- **Light Gray:** #f5f5f5 (slide backgrounds)

### Typography
- **Headings:** 36-48pt, Bold
- **Body Text:** 18-24pt, Regular
- **Captions:** 14-16pt, Light

### Layout Principles
- **Keep it Simple:** Max 3-5 bullet points per slide
- **Use Icons:** Find free icons at flaticon.com or noun project
- **Big Numbers:** Make key metrics 72pt+
- **Consistent Spacing:** Use grids and alignment tools
- **One Idea Per Slide:** Don't cram too much

## Quick Screenshot Guide

### What to Screenshot from Your App

1. **Dashboard Overview**
   - Navigate to http://localhost:3000 (or deployed URL)
   - Login with demo credentials
   - Take full-screen screenshot of dashboard
   - Crop to show: portfolio stats, charts, loan table

2. **Loan Detail Page**
   - Click on any loan
   - Screenshot showing ESG KPIs with progress bars
   - Crop to show metrics and charts

3. **Verification Results**
   - Go to Verifications page
   - Click "View Details" on a verification
   - Screenshot the dialog showing discrepancy
   - Crop to highlight: "Claimed 30%, Verified 22%"

4. **Data Sources**
   - Go to Data Sources page
   - Screenshot showing grid of 50+ sources
   - Crop to show variety of sources

5. **Report Generation**
   - Go to Reports page
   - Screenshot report history
   - Highlight "Generated in 3 seconds"

### Screenshot Tools
- **Windows:** Windows + Shift + S (built-in snipping tool)
- **Mac:** Cmd + Shift + 4
- **Chrome Extension:** Full Page Screen Capture

## Exporting Your Deck

### For Submission
1. **Export as PDF**
   - Google Slides: File → Download → PDF
   - PowerPoint: File → Export → PDF
   - Canva: Share → Download → PDF

2. **Upload to Cloud**
   - Google Drive (get shareable link)
   - Dropbox (get shareable link)
   - Or host on your website

3. **Name It Properly**
   - `ESGLend_Pitch_Deck_LMA_EDGE_2026.pdf`

### For Presentation
- Also save as .pptx or .key for live presenting
- Test animations work
- Practice timing (aim for 5 minutes)

- **0:00-0:30:** Slide 1-2 - Hook with the problem

## Need Help?

If you're stuck on design:
1. Use Canva's "Pitch Deck" templates - they're pre-designed
2. Focus on content first, make it pretty later
3. Keep it simple - white backgrounds with green accents work great
4. The MOST important slide is Slide 6 (Platform Demo) - make it impressive

**Remember:** Judges care more about CONTENT than fancy design. Make sure:
- Numbers are clear
- Problem is urgent
- Solution is obvious
- ROI is compelling
- Demo looks professional
